# Formulate Knowledge

**Version:** `0.1`

Formulate Knowledge helps a learner turn source material into durable retrieval prompts instead of mechanically converting text into flashcards.

It works collaboratively: it maps the material, challenges ambiguity or weak understanding, agrees on useful coverage, drafts representative cards, and produces clean [Mochi Markdown](https://mochi.cards/docs/markdown/basic-formatting/) only after the knowledge is ready.

Invoke it with `$formulate-knowledge` and provide material ranging from a short passage to a large document or URL. It can also critique and repair an existing deck or process a long source incrementally.

See the [skill specification](SKILL.md), [formulation principles](references/formulation-principles.md), and [Mochi Markdown contract](references/mochi-markdown.md).
