# dotskills

A small collection of agent skills I made for myself in the spirit of dotfiles—the personal configuration and small tools people keep to make their computers work the way they like. These skills do the same for the agents I use, carrying useful workflows and judgment between tasks.

Skills are grouped by area and ordered by how often I use them or how much impact I find they have.

## Learning and knowledge

| Name | Description | Version | Rank |
| --- | --- | ---: | :---: |
| [Formulate Knowledge](.agents/skills/learning-and-knowledge/formulate-knowledge/README.md) | Turns source material into well-formulated spaced-repetition knowledge. Default output is Mochi Markdown. | `0.1` | `P5` |

## Ranking

Ranks reflect personal utility rather than objective quality: `P1` is daily or foundational, `P2` is frequent or highly impactful, `P3` is regularly useful, `P4` is occasional or niche, and `P5` is experimental.
